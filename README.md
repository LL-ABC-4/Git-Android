# 이슈 처리 Flow

- []이슈 확인
- []new hotfix branch
  - git branch hotfix/issueName or issueNum
  - git switch hotfix/issueName or issueNum
- []문제 해결(커밋,푸시)
  - 소스 수정 
  - 커밋 git commit
  - 푸시 git push
- []PR 생성
- []이슈에 연결
